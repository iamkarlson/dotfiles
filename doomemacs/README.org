* Doomemacs Configuration 


The doomemacs configuration is organized as follows:

#+begin_src 
doomemacs/
├── init.el                   (Module enablement - 192 lines)
├── config.el                 (Main configuration - 358 lines)
├── packages.el               (Package declarations - 122 lines)
├── custom.el                 (Custom functions - minimal)
├── doomemacs.desktop         (Desktop launcher)
├── docs/                     (Documentation)
├── parts/                    (Modular configuration files)
│   ├── ui.el                (Font and appearance settings)
│   ├── org-roam.el          (Agenda, templates, export, org-roam settings)
│   ├── evil.el              (Evil mode customization)
│   ├── code.el              (Coding settings)
│   ├── dashboard.el         (Dashboard config)
│   ├── magit.el             (Git integration)
│   ├── shortcuts.el         (Keybindings)
│   ├── grep-preview.el      (Grep settings)
│   ├── window-resize.el     (Window management)
│   ├── wayland.el           (Wayland support)
│   ├── structurizr.el       (Structurizr mode)
│   └── human-formatting.el.txt (Archive)
├── snippets/                (Yasnippet snippets)
├── themes/                  (Custom theme files)
└── splash.jpg               (Splash screen image)

#+end_src
